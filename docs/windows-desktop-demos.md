# Windows Desktop Demos

These two demos exist to make the resume's `WPF (MVVM)` and `WinForms` bullets concrete.

## WpfTraceabilityStudio

Positioning:

- Material-image association analysis workstation
- AOI review + engineer decision support
- MVVM structure suitable for upper-machine / internal engineering tooling

What to point out in interviews:

1. Search and data-binding workflow
2. Route replay for station history
3. AI summary plus engineer-side interpretation
4. Command-driven export preview rather than code-behind heavy interaction

## WinFormsEapGateTool

Positioning:

- EAP gate checker for lock/release decisions
- Recipe validation + wafer map review + host command preview
- Fast engineering utility rather than operator dashboard

What to point out in interviews:

1. Mandatory vs optional checks
2. Why a lot can or cannot be released
3. How command preview/logging helps with engineering traceability
4. Why WinForms is still practical for internal factory-side tools

## Environment Note

The current repository was edited from macOS. Source code for both Windows desktop demos is included, but full restore/build/run for these two projects should be done on Windows with the .NET desktop workload available.
